# Segundo-Projeto-HTML

# Apresentação
É um projeto bem simples, apenas para testes e estudos, contendo falhas, e por hora, funcionando apenas para desktop.

# Proposta do Projeto
Na primeira página, temos a opção de 4 filmes. Clicando em qualquer um, você é mandado para uma outra página, onde vê um vídeo com uma cena desse filme, além de um link de uma playlist do Youtube, com outras cenas - focando no gênero do filme.

# Ferramentas
Feito pelo VsCode utilizando HTML e CSS.
